# HabitQuest

HabitQuestは、習慣化をゲーム感覚で楽しめるAndroidアプリです。

毎日の習慣を記録しながら経験値（XP）を獲得し、レベルアップや称号獲得を目指すことで、継続のモチベーション向上をサポートします。

---

## Features

### 習慣管理

* 習慣の追加
* 習慣の削除
* 習慣ごとの詳細画面

### 継続記録

* 毎日の達成記録
* 連続達成日数（Streak）表示
* 達成履歴表示

### RPG要素

* XP（経験値）システム
* レベルシステム
* バッジシステム
* 称号システム
